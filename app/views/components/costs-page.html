При отсутствии мобильной версии дизайна сайта, стоимость работы увеличивается до 10$ / час.

Возможна посадка сайта на CMS WordPress с оформлением личного кабинета, для легкости при изменения сайта, без посторонней помощи

Сайт будет:

  • Хорошо оптимизирован по скорости загрузки (Находится в зелёной зоне по Google PageSpeed)

  • Валиден, т.е. код сайта написан в соответствии с определенными стандартами и не будет содержать ошибок (Проходит проверку на валидность через W3C)
